# Gmail Sync issue on HyperOS
To resolve an sync issue with Gmail on Xiaomi HyperOs, do the following:

1. Remove all Google accounts and add them from the beginning

2. Clear cache data of gmail app.

3. Add google account to gmail app.

4. Open gmail app and go to setting.

5. Then, select the account and go to Notifications.

6. In Notifications>Notifications select All.

7. In Notifications>Inbox notifications, select "Notify for every message".
